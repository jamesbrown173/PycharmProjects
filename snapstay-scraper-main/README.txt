# Instagram Post Downloader

This Python script allows you to download media content (images and videos) from an Instagram post. It supports both single posts and carousels.

## Requirements
- Python 3
- install libraries (`pip install -r requirements.txt`)

## Usage
1. Run the script.  `python3 insta_scraping.py`
2. Input the Instagram post URL when prompted.
3. The script will download the media content and save it in a folder named `data-{username}` in the script's directory.
